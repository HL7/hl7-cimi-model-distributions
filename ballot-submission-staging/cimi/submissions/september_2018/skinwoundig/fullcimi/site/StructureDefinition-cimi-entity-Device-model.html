DeviceModel

 

The official URL for this logical model is: 

http://hl7.org/fhir/us/sw/StructureDefinition/cimi-entity-Device-model

A specific durable physical device used in diagnosis or treatment. The value is the coding for a type of device, for example, a CPAP machine. The same device might be used on multiple patients.

This logical model was published on Fri Aug 17 00:00:00 EDT 2018 as a draft by HL7 Patient Care and Clinical Information Modeling Initiative Work Groups.
