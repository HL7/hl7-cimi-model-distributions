Overview of the Core Package

The CIMI Reference Model Module Core package introduces the foundational classes to support archetyping in ADL. Any class derived from Locatable can be specified as the reference model class that is archetyped in a given ADL archetype. The Archetyped class provides important metadata to support archetyping.
